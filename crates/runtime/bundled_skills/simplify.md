user-invocable: true

Review the recently changed code for opportunities to simplify and improve:

1. Check for code duplication that could be extracted into shared functions
2. Look for overly complex logic that could be simplified
3. Identify unnecessary abstractions or premature optimization
4. Find dead code or unused imports
5. Verify error handling is appropriate (not too much, not too little)
6. Check naming clarity

Fix any issues found directly. Keep changes minimal and focused.
