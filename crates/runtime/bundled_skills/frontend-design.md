Create polished, production-grade frontend interfaces. Generate creative code that avoids generic AI aesthetics.

Guidelines:
- Use modern CSS (grid, flexbox, custom properties, animations)
- Write semantic HTML with proper accessibility (ARIA, focus management)
- Create distinctive visual design — avoid Bootstrap defaults and generic styles
- Use proper responsive design with mobile-first approach
- Include micro-interactions and transitions for polish
- Prefer vanilla JS/CSS unless a framework is specified
- Test across common viewport sizes
